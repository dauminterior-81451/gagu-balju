# AGENTS.md

## 개발자 정보
- 비개발자 출신, ChatGPT로 프롬프트 작성 후 Codex에 붙여넣기 방식으로 작업
- OS: Windows

## 코딩 규칙
- 주석: 한국어
- 함수명: 영문 camelCase
- 변수명: 의미 있게 명명
- 오류 처리: 반드시 포함
- 환경변수(.env) 하드코딩 금지
- 기존 동작 코드 이유 없이 삭제 금지

## 응답 방식
- 결론 먼저  코드  사용법 순서
- 수정 시 파일명과 변경 위치 명시
- 불확실한 로직은 `// TODO: 확인 필요` 주석 표시
- 복붙 가능한 완성 코드로 제공
- 구조 변경 시 계획 먼저 제시, 승인 후 작성

## 작업 범위

| 작업 유형 | 접근 방법 |
|-----------|-----------|
| 변수명주석포맷 | 해당 파일만 최소 수정 |
| 새 함수버그 수정 | 관련 파일 파악 후 작성 |
| 멀티파일 구조 변경 | 변경 계획 먼저 제시 |
